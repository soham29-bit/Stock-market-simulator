# Stock-market-simulator
Terminal-style stock trading simulator with live price charts, portfolio tracker &amp; buy/sell engine

# 📈 SYS STOCKS — Live Trading Simulator

A real-time stock market simulator built with Python using only **NumPy**, **Pandas**, and **Matplotlib**.

## Features
- Live price simulation with auto-refresh every 3 seconds
- Buy & Sell stocks with balance tracking
- Portfolio view with unrealized P/L
- Live price history charts for all 4 symbols

## Run it
```bash
pip install numpy pandas matplotlib
python sys_stocks.py
```
